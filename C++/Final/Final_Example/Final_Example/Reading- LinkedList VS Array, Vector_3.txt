- 연결리스트

 각 요소가 포인터로 연결되어있음. 

	단일 연결리스트일 경우 단방향
	이중 연결리스트일 경우 쌍방향
	순환 단일 연결리스트일 경우 단방향에 끝부분이 처음과 연결되어 있음
	순환 이중 연결리스트일 경우 쌍방향에 처음이 끝 부분과, 끝부분이 처음과 연결되어있음


 


- 배열, 벡터
 
 각 요소가 연속된 공간에 저장되어 있음.


공통점:

차이점: 
 연결리스트는 임의 접근(random access) 불가능, 이는 각 요소들이 포인터로 연결되어 있어서.

 배열, 벡터는 "연속된" 공간을 점유하고 있어서 이 임의접근이 가능.
